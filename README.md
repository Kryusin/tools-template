# tools-template

## 概要 - overview
このプロジェクトは、各種ツールのテンプレートとして扱うリポジトリです。

This project is a repository to be treated as a template for various tools.

## 各種ツール
- biome
    - biome.json
- cspell
    - cspell.config.cjs
    - cspell.config.yaml
    - cspell.json
- eslint
- prittier
- tailwind
- tsconfig
